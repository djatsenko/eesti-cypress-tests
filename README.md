# Eesti.ee Cypress tests

## Kirjeldus

See projekt sisaldab Cypress-i UI teste Eesti.ee veebilehe jaoks. Testid katavad järgmised stsenaariumid:
- Avalehe otsing
- Lehe navigatsioon
- Kontaktivormi olemasolu
- Keele vahetamine

## Käivitamine

1. `npm install`
2. `npx cypress open`
